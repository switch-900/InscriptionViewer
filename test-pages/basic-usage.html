1. Simple Array of Inscription IDs

The most basic usage - just pass an array of inscription IDs:

import { InscriptionGallery } from 'bitcoin-inscription-viewer';

const inscriptionIds = [
  'dca3da701a2607de6c89dd0bfe6106532dcefe279d13b105301a2d85eb4ffaafi0',
  '0e50a465fc0ca415f3cb8a4aac1555b12a4bf3f33bc039f2a4d39f809e83af7ai0',
  '934905624f847731e7f173ba70bfa3a1389b0a7fe2a4ffce8793eef2730b9ab9i0'
];

<InscriptionGallery
  inscriptionIds={inscriptionIds}
  columns={3}
  cardSize={200}
  showIndex={true}
  enableModal={true}
/>

2. API Response Format with IDs

Handle API responses from endpoints like /inscriptions that return { ids: [...] }:

// API response from /inscriptions endpoint
const apiResponse = {
  ids: [
    'dca3da701a2607de6c89dd0bfe6106532dcefe279d13b105301a2d85eb4ffaafi0',
    '0e50a465fc0ca415f3cb8a4aac1555b12a4bf3f33bc039f2a4d39f809e83af7ai0',
    '934905624f847731e7f173ba70bfa3a1389b0a7fe2a4ffce8793eef2730b9ab9i0'
  ]
};

<InscriptionGallery
  inscriptions={apiResponse}
  columns={3}
  cardSize={200}
  showHeaders={true}
  showControls={true}
/>

3. API Response with Children Objects

Handle API responses from endpoints like /children that return full objects:

// API response from /children endpoint
const apiResponse = {
  children: [
    {
      id: 'dca3da701a2607de6c89dd0bfe6106532dcefe279d13b105301a2d85eb4ffaafi0',
      number: 1,
      contentType: 'text/plain'
    },
    {
      id: '0e50a465fc0ca415f3cb8a4aac1555b12a4bf3f33bc039f2a4d39f809e83af7ai0', 
      number: 2,

üöÄ Quick Start

Get started in under 60 seconds:

# 1. Install the package
npm install bitcoin-inscription-viewer

# 2. Import and use
import { InscriptionGallery } from 'bitcoin-inscription-viewer';
import 'bitcoin-inscription-viewer/dist/style.css';

function MyApp() {
  const inscriptions = ['your-inscription-id-here'];
  
  return (
    <InscriptionGallery
      inscriptionIds={inscriptions}
      columns={3}
      cardSize={200}
      enableModal={true}
    />
  );
}
